신성쌤짱

#step_1
제일 먼저 대장균의 생장곡선을 출력하는 코드를 작성했다.
과정: 신뢰도 있는 자료를 바탕으로 식품의 부패 초기단계의 균 수를 5*10**6 CFU/g 로 설정했다. 편의를 위해 코드에서는 그냥 10**6 으로 설정했다.
그리고 2학년때 생장곡선에 대해 수학1 교과에서 탐구했던 내용 + 로지스틱 생장곡선에 대한 추가적 조사를 바탕으로 최대 균 수(환경수용력)와 생장곡선 함수식을 설정했다.
시행착오: 처음엔 로그스케일로 출력했으나 사용자가 로그스케일로 균 수의 기하급수적 증가를 체감하지 못 할 것 같다는 피드백을 바탕으로, 직관적 위험도를 보여주기 위해 그래프를 일반 스케일로 수정했다. 

#step_2
그다음 1단계 코드를 바탕으로 환경 변수를 직접 조절할 수 있는 생장곡선 시뮬레이터 웹앱을 제작했다.
과정: 슬라이더를 추가해 환경 변수 세 가지(온도, 습도, 보관 시간)을 조절할 수 있도록 했다. 또한 균 수가 기준치에 도달하는 시간을 계산해 화면에 띄워 예상 소비기한과 보관 환경 맞춤 조언을 제공했다.
웹앱으로 배포하기 위해 코드 형식을 수정하고 깃허브를 이용해 공공 배포가 가능한 형태로 만들었다.
시행착오: 
참고: https://pmp.errc.ars.usda.gov/WaterActivity.aspx (미국농무부 PMP)

#step_3
마지막으로 내가 원하는 음식 종류를 고를 수 있는 기능을 추가해 완성했다.
과정: 음식 종류에 따라 변수 세팅을 다르게 했다. 각 음식의 특성과 수분활성도에 따른
